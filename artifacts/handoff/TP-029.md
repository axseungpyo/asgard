## Objective

한 에이전트(Heimdall)의 결과를 다른 에이전트(Brokkr)의 TP에 **자동으로 반영**하여,
에이전트 간 **컨텍스트를 공유**하는 메커니즘을 구축한다.

```
Heimdall: TP-032 (UI 분석) 완료
  ↓ RP-032 결과 자동 추출
Brokkr: TP-033 (프론트 구현)에 Heimdall 분석 결과 주입
  ↓
Brokkr가 분석 결과를 참고하여 구현
```

## Context

### 현재 상태
- 에이전트는 독립적으로 실행되며 서로의 결과를 모름
- RP(Result Packet)가 파일로 존재하지만 자동 연결 없음
- PlanStep에 dependsOn이 있지만 결과 전달 기능 없음

### 목표
1. PlanStep에 **contextFrom** 필드 — 이전 step 결과를 참조
2. **AgentContextService** — RP 파싱 + 컨텍스트 요약 추출
3. PlannerUseCase가 step 실행 시 **이전 결과를 input에 주입**
4. 컨텍스트 공유 이벤트 발행

### 3. core/use-cases/plan/ContextService.ts (신규, ~60줄)

```typescript
export class ContextService {
  constructor(
    private readonly fileSystem: IFileSystem,
    private readonly projectRoot: string,
  ) {}

  /**
   * 완료된 step의 결과에서 컨텍스트를 추출한다.
   * - RP 파일이 있으면 Summary + AC 결과 추출
   * - step.result가 있으면 그대로 사용
   * - 최대 2000자로 잘라서 반환
   */
  async extractContext(step: PlanStep): Promise<string> {
    // 1. RP 파일 확인 (step.result에서 RP-NNN 패턴 추출)
    const rpMatch = step.result?.match(/RP-\d+/);
    if (rpMatch) {
      try {
        const rpPath = path.join(this.projectRoot, "artifacts", "handoff", `${rpMatch[0]}.md`);
        const content = await this.fileSystem.readFile(rpPath);
        return this.summarizeRP(content);
      } catch {
        // RP 파일 없으면 step.result 사용
      }
    }

    // 2. step.result 사용
    if (step.result) {
      return step.result.slice(0, 2000);
    }

    return "";
  }

  private summarizeRP(content: string): string {
    // Summary 섹션 추출
    const summaryMatch = content.match(/## Summary\n([\s\S]*?)(?=\n## |$)/);
    const acMatch = content.match(/## Acceptance Criteria\n([\s\S]*?)(?=\n## |$)/);

    const parts: string[] = [];
    if (summaryMatch) parts.push(summaryMatch[1].trim());
    if (acMatch) parts.push(acMatch[1].trim());

    return parts.join("\n\n").slice(0, 2000) || content.slice(0, 2000);
  }
}
```

### 4. PlannerUseCase 수정 (~30줄)

executeStep에서 contextFrom 처리:

```typescript
constructor(
  // ... 기존
  private readonly contextService: ContextService,
) {}

private async executeStep(plan: ExecutionPlan, step: PlanStep): Promise<void> {
  // 컨텍스트 주입
  if (step.contextFrom?.length) {
    const contexts: string[] = [];
    for (const fromOrder of step.contextFrom) {
      const fromStep = plan.steps.find(s => s.order === fromOrder);
      if (fromStep?.status === "completed") {
        const ctx = await this.contextService.extractContext(fromStep);
        if (ctx) {
          contexts.push(`[Step ${fromOrder} 결과]\n${ctx}`);
          this.eventBus.publish({
            type: CONTEXT_SHARED_EVENT,
            timestamp: Date.now(),
            payload: {
              fromStep: fromOrder,
              toStep: step.order,
              fromAgent: fromStep.input.agent || "unknown",
              toAgent: step.input.agent || "unknown",
              contextSummary: ctx.slice(0, 200),
              planId: plan.id,
            },
          });
        }
      }
    }
    if (contexts.length > 0) {
      step.contextData = contexts.join("\n\n---\n\n");
      // input에 컨텍스트 추가
      step.input = { ...step.input, previousContext: step.contextData };
    }
  }

  // ... 이후 기존 실행 로직
}
```
