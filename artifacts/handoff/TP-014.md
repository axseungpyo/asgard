# TP-014: Asgard Night Sky 테마 — 대시보드 가독성 개선

## Metadata
- Agent Target: codex
- Complexity: complex
- Priority: P1
- Created: 2026-03-09

## Objective

Yggdrasil 대시보드의 색상 체계를 "Asgard Night Sky" 테마로 교체하여, 어두운 배경 위 컨텐츠 가독성을 WCAG AA 기준 이상으로 개선한다.

## Scope In

### 1. globals.css 색상 토큰 교체

현재 → 변경:

```
/* 배경: 완전한 검정 → 깊은 남색(deep indigo/navy) */
--color-bg-primary:   #09090b → #0c0f1a     /* 깊은 남색 */
--color-bg-secondary: #111113 → #131729     /* 약간 밝은 남색 */
--color-bg-tertiary:  #1a1a1e → #1a1f35     /* 카드/패널 배경 */
--color-bg-elevated:  #27272a → #242b45     /* 팝업/호버 배경 */
--color-border:       #27272a → #2a3155     /* 테두리: 남색 톤 */

/* 상태 색상: 채도/밝기 약간 상향 */
--color-status-idle:    #a1a1aa → #9ca3af   /* 더 밝은 회색 */
--color-status-running: #a78bfa → #b197fc   /* 더 밝은 바이올렛 */
--color-status-done:    #a3e635 → #86efac   /* 더 부드러운 민트 그린 */
--color-status-review:  #f0abfc → #f0abfc   /* 유지 */
--color-status-draft:   #71717a → #94a3b8   /* slate 톤으로 밝기 증가 */

/* 추가 토큰 */
--color-text-primary:   #f1f5f9   /* slate-100: 주요 텍스트 */
--color-text-secondary: #cbd5e1   /* slate-300: 보조 텍스트 */
--color-text-muted:     #94a3b8   /* slate-400: 약한 텍스트 (최소 대비) */
--color-text-faint:     #64748b   /* slate-500: 가장 약한 텍스트 */
```

body 배경색과 기본 텍스트 색상도 함께 변경:
```
body {
  background-color: #0c0f1a;
  color: #e2e8f0;  /* slate-200 */
}
```

scrollbar, selection 색상도 남색 톤으로 조정.

### 2. 컴포넌트 텍스트 색상 일괄 교체

아래 패턴으로 대시보드 전체 컴포넌트의 텍스트 색상을 교체:

| 현재 | 변경 | 용도 |
|------|------|------|
| `text-zinc-100` | `text-slate-100` | 주요 제목/이름 |
| `text-zinc-200` | `text-slate-200` | 활성 탭/선택된 항목 |
| `text-zinc-300` | `text-slate-300` | 본문 텍스트 |
| `text-zinc-400` | `text-slate-400` | 보조 레이블 |
| `text-zinc-500` | `text-slate-400` | 섹션 헤더 (한 단계 밝게) |
| `text-zinc-600` | `text-slate-500` | 타임스탬프, 부가 정보 (두 단계 밝게) |
| `text-zinc-700` | `text-slate-500` | 빈 상태 메시지 (세 단계 밝게, 핵심 수정) |

배경 관련:
| 현재 | 변경 | 용도 |
|------|------|------|
| `bg-zinc-950/*` | `bg-[#0c0f1a]/*` 또는 `bg-bg-primary/*` | 투명 오버레이 |
| `bg-zinc-900/*` | `bg-bg-secondary/*` | 카드 내부 |
| `bg-zinc-800/*` | `bg-bg-tertiary/*` | 호버, 헤더 행 |
| `border-zinc-800/*` | `border-[--color-border]/*` | 테두리 |
| `border-zinc-700` | `border-slate-700` | 버튼 테두리 |

### 3. 에이전트 색상 미세 조정

에이전트 메인 색상은 유지하되, 카드 배경에 에이전트 색상의 미세한 글로우 효과 추가:
```css
/* AgentCard에 에이전트 색상 배경 글로우 (5% 투명도) */
background: linear-gradient(135deg, rgba(agent-color, 0.05), transparent);
```

### 5. 대비 검증

교체 후 최소 대비율 확인 (WCAG AA: 4.5:1):
- `text-slate-100` (#f1f5f9) on `bg-primary` (#0c0f1a) → ~15:1 (PASS)
- `text-slate-300` (#cbd5e1) on `bg-primary` (#0c0f1a) → ~11:1 (PASS)
- `text-slate-400` (#94a3b8) on `bg-primary` (#0c0f1a) → ~6.5:1 (PASS)
- `text-slate-500` (#64748b) on `bg-primary` (#0c0f1a) → ~4.5:1 (PASS, 최소 기준)
- `text-slate-500` (#64748b) on `bg-secondary` (#131729) → ~4.2:1 (근접, 허용)

## Acceptance Criteria
1. `globals.css`의 색상 토큰이 Asgard Night Sky 테마로 교체되어 있다
2. 대시보드 전체에서 `text-zinc-700` 클래스가 0개이다 (`grep -r "text-zinc-700"` 결과 없음)
3. 대시보드 전체에서 `text-zinc-600` 클래스가 0개이다 (모두 `text-slate-500` 이상으로 교체)
4. `lib/constants.ts`의 STATUS_CONFIG, TASK_STATUS_CONFIG 색상이 업데이트되어 있다
5. `npm test` 전체 테스트 PASS
6. TypeScript 컴파일 에러가 없다
7. `npm run build` 성공

## Implementation Notes
- zinc → slate 계열로 전환하는 이유: slate는 남색 기반 회색이라 깊은 남색 배경과 조화됨. zinc는 순수 회색이라 검정 배경에서 차가운 느낌
- 에이전트 색상(odin=#d97757, brokkr=#10a37f, heimdall=#4285f4, loki=#a855f7)은 변경하지 않음 — 이미 남색 배경에서 충분한 대비
- `replace-all` 패턴으로 일괄 교체 시 각 파일별 확인 필요 (일부 컴포넌트에서 의도적으로 다른 값 사용 가능)
- body 색상 변경 후 auth 모달, API keys 모달의 배경도 함께 확인
