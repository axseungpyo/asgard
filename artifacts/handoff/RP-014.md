## Known Issues
- `npx tsc --noEmit`는 `.next/types/**/*.ts`를 포함하는 현재 `tsconfig.json` 때문에 빌드 전 단독 실행 시 `.next/types/cache-life.d.ts not found`로 실패할 수 있다. 이번 검증에서는 `npm run build` 이후 재실행하여 실제 타입 오류가 없음을 확인했다.
- `npm run build` 시 Next.js가 `/Users/hongseungpyo/package-lock.json`을 추가 lockfile로 감지해 workspace root 경고를 출력한다. 빌드는 정상 완료됐다.
